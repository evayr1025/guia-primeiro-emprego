# guia-primeiro-emprego
# Guia Prático: Como Conseguir o Primeiro Emprego

Este projetotem como objativo ajudar pessoas que estão em busca do
primeiro emprego, oferecendo dicas práticas, orientações e passos
importante para entrar no mercado de trabalho

---

## Sobre o Projeto

Muitas pessoas enfrentam dificuldades ao buscar o primeiro 
emprego por falta de experiência. Este guia foi criado para
facilitar esse processo de forma simples e direta

---

## Objetivos

-  Ajudar iniciantes no mercado de trabalho
-  Ensinar como montar um currículo
-  Dar dicas de comportamento em uma entrevista
-  Incentivar o desenvolvimento pessaol e profissional

## Engenharia de Prompts

### Prompt 1:
**Como fazer um currículo para primeiro emprego?**
 Um currículo para primeiro emprego deve ser simples 
 e organizado, contendo dados pessoais, objetivo
 profissional, formação acadêmica e habilidades

---

### Perguntas adicionais:

- Como se preparar para entrevista?
- O que colocar no currículo sem experiência?
- Quais erros evitar ao procurar emprego?
- Como se comportar em uma entrevista?

  ---

  ## Resumo do Tema

  ### O que é primeiro emprego
  ### Dificuldades
  ### Dicas Principais

  Conseguir o primeiro emprego pode ser desafiador
  por falta de experiência, mas é possível atráves
  de preparo, organização e desenvolvimento de
  habilidades básicas como comunicação e responsabilidade

  ---

  ## Glossário

   -**Currículo:** documento com suas informações
  profissionais
  -**Entrevista:** Conversa com o recrutador
  -**Soft skills:** Habilidade comportamentais
  -**Hard skills:** Habilidades técnicas (informática, cursos)
  -**Recrutador:** Pessoa responsável pela contratação
